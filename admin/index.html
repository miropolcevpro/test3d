Открыть сайт Валидатор палитры

Вход

Введите логин и пароль. Админка работает через backend API (JWT).

Backend URL задаётся в admin/config.js (переменная API_BASE_URL).

Формы

Список форм загружается через backend: GET /api/shapes. Палитра формы: GET /api/palettes/<shapeId>. Если палитры нет (404 в бакете) — backend вернёт пустой шаблон.

Текстуры формы
