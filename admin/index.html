Загрузка

Загрузка выполняется напрямую в Object Storage по presigned URL (без проксирования через backend). Для этого в бакете должен быть разрешён PUT в CORS для домена https://miropolcevpro.github.io.

Правильная структура: surfaces/<shapeId>/<textureId>/1k/<textureId>_albedo.webp (и аналогично для normal/roughness/height/ao).
Минимум: albedo + normal + roughness + height. AO опциональна.
Подсказка по именам: файлы должны содержать суффиксы _albedo, _normal, _roughness, _height, _ao.
Параметры загрузки
Файлы
Поддерживаются два режима ZIP:
  • Умная сборка: ZIP уже содержит структуру surfaces/<shapeId>/<textureId>/<quality>/... — тогда textureId вводить не нужно, можно загрузить сразу много текстур.
  • Простой: ZIP содержит только файлы — тогда укажите textureId и quality в форме. Имена файлов должны содержать суффиксы _albedo, _normal, _roughness, _height (опционально _ao).
  • Нестандартные имена: если в структурированном ZIP файлы названы без суффиксов, админка предложит окно сопоставления карт (вы вручную выберете, какой файл = albedo/normal/roughness/height).

Превью: albedo (из палитры). «После» учитывает дефолты палитры + переопределения текстуры.
Как работает: если параметр не задан у текстуры, сайт берёт дефолт из «Настройки палитры». Для “чистого” сброса используйте кнопку «Сбросить к дефолтам палитры» — она удалит переопределения.
Подсказки по настройкам (что на что влияет)
  • Плитка выглядит слишком блестящей/"пластиковой" → увеличьте roughnessMult и/или уменьшите specStrength.
  • Тёмная плитка выбеливается (пересвет) → уменьшите exposureMult и слегка снизьте contrast.
  • Рельеф слишком "шумный" → уменьшите normalScale (часто 0.6–1.0 достаточно).
  • Появляются неестественные тени → уменьшите bumpScale (обычно 0.2–0.8).
  • Узор выглядит мелко/крупно → uvScale (меньше 1 — крупнее, больше 1 — мельче).
  • Насыщенность держите умеренной: в AR завышенная saturation чаще выглядит нереалистично.
Массовая настройка текстур
Применение: вы можете настроить параметры сразу для нескольких текстур. Поля с выключенным переключателем «Применять» не изменятся.
Рекомендация: начните с «Вставить дефолты палитры», затем аккуратно корректируйте.
Подсказки по массовым правкам
  • Если хотите унифицировать вид всей формы, выберите цель «Всем текстурам формы» и выставьте базовые значения (обычно roughnessMult, specStrength, normalScale).
  • Для аккуратной коррекции цветов начните с небольших шагов по exposureMult (±0.05) и contrast (±0.03).
  • Если сомневаетесь — используйте «Сбросить переопределения» и вернитесь к дефолтам палитры.
Если выключено — tileSizeM у текстур не трогаем.
