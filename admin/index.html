Вход

Введите логин и пароль. Админка работает через backend API (JWT).

Backend URL задаётся в admin/config.js (переменная API_BASE_URL).

Контент

Данные загружаются через backend API (JWT): GET /api/shapes, GET /api/palettes/<shapeId>. Если палитры нет (404 в бакете), backend возвращает пустой шаблон.

Текстуры формы

В палитре пока нет текстур.

Загрузка

Здесь будет загрузка ZIP и отдельных файлов с подсказками, прогрессом и отчётом. В текущей итерации мы подготовили экран формы и навигацию.

Рекомендации по структуре: surfaces/<shapeId>/<textureId>/1k/... и (опционально) 2k/.... Обязательные карты: albedo, normal, roughness, height.

Настройки палитры

Здесь будут дефолтные настройки масштаба и цветопередачи для формы (safe-fallback на сайте).

Примеры: default tile size, uvScale, exposure/contrast/saturation, roughness/spec strength.
